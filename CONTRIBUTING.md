# Contribuir a Agora

1. Haz un fork del repositorio.
2. Crea o modifica un banco dentro de `banks/`.
3. Actualiza `catalog.json` si añades un banco nuevo.
4. Comprueba que el banco tiene `scope` y referencias canónicas válidas.
5. Comprueba que cada línea de `questions.jsonl` es JSON válido y tiene `refs`.
6. Abre un pull request explicando qué has cambiado y de dónde procede el contenido.

No subas contenido si no tienes derecho a usarlo.

Los `tags` ayudan a buscar, pero no sustituyen a las referencias canónicas. Si falta una ley, artículo, oposición o tema, añádelo primero en `refs/` o `syllabi/`.
