# Seguridad

No subas secretos a este repositorio.

No deben publicarse:

- claves API;
- tokens de GitHub;
- credenciales de WordPress;
- volcados de base de datos;
- copias de seguridad privadas;
- datos personales;
- contenido sobre el que no tengas derechos.

Si detectas un secreto publicado por error, elimina el contenido de inmediato y rota la clave afectada. Para avisos de seguridad, abre un issue sin incluir el secreto en claro.
