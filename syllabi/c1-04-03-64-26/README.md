# C1-04-03::64-26

Representación estructurada de la convocatoria como lista de referencias normativas exigidas.

## Criterio

La convocatoria no contiene preguntas. La convocatoria contiene refs:

```text
convocatoria.refs ∋ es-ley-39-2015::titulo-iv
pregunta.refs ∋ es-ley-39-2015::titulo-iv
```

Por tanto, una pregunta es compatible con la convocatoria si intersecta con sus refs.
